# 大方广佛华严经卷第八

*《大方广佛华严经》卷第八*

于阗国三藏实叉难陀奉 制译

*于阗国三藏法师实叉难陀奉皇帝命令翻译*

华藏世界品第五之一

*华藏世界品第五章之一*

尔时，普贤菩萨复告大众言：“诸佛子！此华藏庄严世界海，是毗卢遮那如来往昔于世界海微尘数劫修菩萨行时，一一劫中亲近世界海微尘数佛，一一佛所净修世界海微尘数大愿之所严净。诸佛子！此华藏庄严世界海，有须弥山微尘数风轮所持。其最下风轮，名：平等住，能持其上一切宝焰炽然庄严。次上风轮，名：出生种种宝庄严，能持其上净光照耀摩尼王幢。次上风轮，名：宝威德，能持其上一切宝铃。次上风轮，名：平等焰，能持其上日光明相摩尼王轮。次上风轮，名：种种普庄严，能持其上光明轮华。次上风轮，名：普清净，能持其上一切华焰师子座。次上风轮，名：声遍十方，能持其上一切珠王幢。次上风轮，名：一切宝光明，能持其上一切摩尼王树华。次上风轮，名：速疾普持，能持其上一切香摩尼须弥云。次上风轮，名：种种宫殿游行，能持其上一切宝色香台云。诸佛子！彼须弥山微尘数风轮，最在上者，名：殊胜威光藏，能持普光摩尼庄严香水海；此香水海有大莲华，名：种种光明蕊香幢。华藏庄严世界海，住在其中，四方均平，清净坚固；金刚轮山，周匝围绕；地海众树，各有区别。”

*那时，普贤菩萨又告诉大众说：“各位佛子！这个华藏庄严世界海，是毗卢遮那如来往昔在世界海微尘那么多数的劫数中修行菩萨道时，在每一个劫数中亲近世界海微尘那么多数的佛，在每一尊佛那里清净地修行世界海微尘那么多数的大愿所庄严清净的。各位佛子！这个华藏庄严世界海，有须弥山微尘那么多数的风轮所支撑。它最下面的风轮，名叫：平等住，能支撑它上面的一切宝焰炽然的庄严。其次上面的风轮，名叫：出生种种宝庄严，能支撑它上面的净光照耀的摩尼王幢。其次上面的风轮，名叫：宝威德，能支撑它上面的一切宝铃。其次上面的风轮，名叫：平等焰，能支撑它上面的日光明相的摩尼王轮。其次上面的风轮，名叫：种种普庄严，能支撑它上面的光明轮华。其次上面的风轮，名叫：普清净，能支撑它上面的一切华焰师子座。其次上面的风轮，名叫：声遍十方，能支撑它上面的一切珠王幢。其次上面的风轮，名叫：一切宝光明，能支撑它上面的一切摩尼王树华。其次上面的风轮，名叫：速疾普持，能支撑它上面的一切香摩尼须弥云。其次上面的风轮，名叫：种种宫殿游行，能支撑它上面的一切宝色香台云。各位佛子！那些须弥山微尘那么多数的风轮，最上面的那个，名叫：殊胜威光藏，能支撑普光摩尼庄严的香水海；这个香水海里有一朵大莲华，名叫：种种光明蕊香幢。华藏庄严世界海，就住在这里面，四方平均平坦，清净坚固；金刚轮山，周遍围绕；大地、海洋、各种树木，各有区别。”*

是时，普贤菩萨欲重宣其义，承佛神力，观察十方而说颂言：

*这时，普贤菩萨想再宣说它的义理，承受佛的神力，观察十方而说偈颂道：*

“世尊往昔于诸有，　　微尘佛所修净业，
　故获种种宝光明，　　华藏庄严世界海。
　广大悲云遍一切，　　舍身无量等刹尘，
　以昔劫海修行力，　　今此世界无诸垢。
　放大光明遍住空，　　风力所持无动摇，
　佛藏摩尼普严饰，　　如来愿力令清净。
　普散摩尼妙藏华，　　以昔愿力空中住，
　种种坚固庄严海，　　光云垂布满十方。
　诸摩尼中菩萨云，　　普诣十方光炽然，
　光焰成轮妙华饰，　　法界周流靡不遍。
　一切宝中放净光，　　其光普照众生海，
　十方国土皆周遍，　　咸令出苦向菩提。
　宝中佛数等众生，　　从其毛孔出化形，
　梵主帝释轮王等，　　一切众生及诸佛。
　化现光明等法界，　　光中演说诸佛名，
　种种方便示调伏，　　普应群心无不尽。
　华藏世界所有尘，　　一一尘中见法界，
　宝光现佛如云集，　　此是如来刹自在。
　广大愿云周法界，　　于一切劫化群生，
　普贤智地行悉成，　　所有庄严从此出。”

*“世尊往昔在各种存在中，在微尘那么多数的佛那里修行清净之业，所以获得各种宝光明，成就了华藏庄严世界海。广大的悲心之云遍及一切，舍弃的身体无量，等同于刹土的微尘，用往昔在劫海中修行的力量，现在这个世界没有各种尘垢。放出大光明遍布安住在空中，被风的力量所支撑而没有动摇，佛藏摩尼宝珠普遍庄严装饰，如来的愿力让它清净。普遍散布摩尼宝珠的美妙宝藏之华，用往昔的愿力安住在空中，各种坚固的庄严之海，光云垂挂分布充满十方。在各种摩尼宝珠中有菩萨如同云彩，普遍前往十方世界，光芒炽盛，光焰形成轮子，被美妙的华朵装饰，在法界中周流，没有不遍及的。在一切宝物中放出清净的光明，光明普遍照耀众生之海，十方国土都周遍了，都让他们脱离苦难，趣向菩提。宝物中的佛的数量等同于众生，从他们的毛孔中化现出各种形体，大梵天主、帝释天、转轮圣王等等，一切众生以及各位佛。化现的光明等同于法界，在光明中演说各位佛的名号，用各种方便法门开示调伏，普遍感应众生的心而没有不穷尽的。华藏世界所有的微尘，在每一个微尘中都看见法界，宝光中显现的佛如同云彩聚集，这是如来刹土的自在。广大的誓愿之云周遍法界，在一切劫数中教化众生，普贤菩萨的智慧之地和行门都成就了，所有的庄严都由此而出。”*

尔时，普贤菩萨复告大众言：“诸佛子！此华藏庄严世界海大轮围山，住日珠王莲华之上，栴檀摩尼以为其身，威德宝王以为其峰，妙香摩尼而作其轮，焰藏金刚所共成立，一切香水流注其间；众宝为林，妙华开敷，香草布地，明珠间饰，种种香华处处盈满；摩尼为网，周匝垂覆……。如是等，有世界海微尘数众妙庄严。”

*那时，普贤菩萨又告诉大众说：“各位佛子！这个华藏庄严世界海的大轮围山，安住在日珠王莲华之上，用栴檀摩尼作为它的身体，用威德宝王作为它的山峰，用美妙的香摩尼作为它的轮子，由焰藏金刚共同构成，一切香水都流淌在其中；各种宝物作为树林，美妙的花朵都开放，香草铺在地上，明珠交错装饰，各种香华处处都充满了；用摩尼宝珠作为网，周遍垂挂覆盖……。像这样等等，有世界海微尘那么多数的美妙庄严。”*

尔时，普贤菩萨欲重宣其义，承佛神力，观察十方而说颂言：

*那时，普贤菩萨想再宣说它的义理，承受佛的神力，观察十方而说偈颂道：*

“世界大海无有边，　　宝轮清净种种色，
　所有庄严尽奇妙，　　此由如来神力起。
　摩尼宝轮妙香轮，　　及以真珠灯焰轮，
　种种妙宝为严饰，　　清净轮围所安住。
　坚固摩尼以为藏，　　阎浮檀金作严饰，
　舒光发焰遍十方，　　内外映彻皆清净。
　金刚摩尼所集成，　　复雨摩尼诸妙宝，
　其宝精奇非一种，　　放净光明普严丽。
　香水分流无量色，　　散诸华宝及栴檀，
　众莲竞发如衣布，　　珍草罗生悉芬馥。
　无量宝树普庄严，　　开华发蕊色炽然，
　种种名衣在其内，　　光云四照常圆满。
　无量无边大菩萨，　　执盖焚香充法界，
　悉发一切妙音声，　　普转如来正法轮。
　诸摩尼树宝末成，　　一一宝末现光明，
　毗卢遮那清净身，　　悉入其中普令见。
　诸庄严中现佛身，　　无边色相无央数，
　悉往十方无不遍，　　所化众生亦无限。
　一切庄严出妙音，　　演说如来本愿轮，
　十方所有净刹海，　　佛自在力咸令遍。”

*“世界的大海没有边际，宝轮清净，有各种颜色，所有的庄严都奇妙，这是由于如来的神力而兴起的。摩尼宝轮、妙香轮，以及真珠灯焰轮，用各种美妙的宝物作为庄严的装饰，是清净的轮围山所安住的。坚固的摩尼宝作为宝藏，用阎浮檀金作为庄严的装饰，舒展光明，发出火焰，遍及十方世界，内外映照，都清净。由金刚摩尼所构成，又降下摩尼各种美妙的宝物，宝物精奇不止一种，放出清净的光明，普遍庄严华丽。香水分流，有无量的颜色，散布着各种华宝和栴檀，众多的莲花竞相开放，如同衣服一样铺陈，珍奇的草罗列生长，都芬芳馥郁。无量的宝树普遍庄严，开花吐蕊，色彩炽盛，各种名贵的衣服在其中，光云四面照耀，永远圆满。无量无边的大菩萨，执着伞盖，焚烧着香，充满法界，都发出一切美妙的声音，普遍转动如来的正法轮。各种摩尼树由宝末构成，每一个宝末都显现光明，毗卢遮那佛的清净身体，都进入其中，普遍让众生看见。在各种庄严中显现佛身，无边的色相无数，都前往十方世界而无不遍及，所教化的众生也无限。一切庄严都发出美妙的声音，演说如来的本愿之轮，十方世界所有的清净刹海，佛的自在之力都让它遍及。”*

尔时，普贤菩萨复告大众言：“诸佛子！此世界海大轮围山内所有大地，一切皆以金刚所成，坚固庄严，不可沮坏；清净平坦，无有高下；摩尼为轮，众宝为藏；一切众生，种种形状；诸摩尼宝，以为间错；散众宝末，布以莲华；香藏摩尼，分置其间；诸庄严具，充遍如云，三世一切诸佛国土所有庄严而为校饰；摩尼妙宝以为其网，普现如来所有境界，如天帝网于中布列。诸佛子！此世界海地，有如是等世界海微尘数庄严。”

*那时，普贤菩萨又告诉大众说：“各位佛子！这个世界海大轮围山内所有的大地，一切都用金刚构成，坚固庄严，不可损坏；清净平坦，没有高低；用摩尼宝作为轮，用各种宝物作为宝藏；一切众生，有各种形状；用各种摩尼宝，作为交错的装饰；散布各种宝末，铺上莲华；用香藏摩尼，分置在其中；各种庄严的用具，充满如同云彩，用三世一切诸佛国土所有的庄严作为装饰；用摩尼妙宝作为它的网，普遍显现如来所有的境界，如同天帝的网在其中排列。各位佛子！这个世界海的地，有像这样等等的世界海微尘那么多数的庄严。”*

尔时，普贤菩萨欲重宣其义，承佛神力，观察十方而说颂言：

*那时，普贤菩萨想再宣说它的义理，承受佛的神力，观察十方而说偈颂道：*

“其地平坦极清净，　　安住坚固无能坏，
　摩尼处处以为严，　　众宝于中相间错。
　金刚为地甚可悦，　　宝轮宝网具庄严，
　莲华布上皆圆满，　　妙衣弥覆悉周遍。
　菩萨天冠宝璎珞，　　悉布其地为严好，
　栴檀摩尼普散中，　　咸舒离垢妙光明。
　宝华发焰出妙光，　　光焰如云照一切，
　散此妙华及众宝，　　普覆于地为严饰。
　密云兴布满十方，　　广大光明无有尽，
　普至十方一切土，　　演说如来甘露法。
　一切佛愿摩尼内，　　普现无边广大劫，
　最胜智者昔所行，　　于此宝中无不见。
　其地所有摩尼宝，　　一切佛刹咸来入，
　彼诸佛刹一一尘，　　一切国土亦入中。
　妙宝庄严华藏界，　　菩萨游行遍十方，
　演说大士诸弘愿，　　此是道场自在力。
　摩尼妙宝庄严地，　　放净光明备众饰，
　充满法界等虚空，　　佛力自然如是现。
　诸有修治普贤愿，　　入佛境界大智人，
　能知于此刹海中，　　如是一切诸神变。”

*“那里的地平坦极为清净，安住坚固，无人能破坏，摩尼宝处处都作为庄严，各种宝物在其中交错。金刚作为地，非常令人喜悦，宝轮和宝网都具足庄严，莲华铺在上面都圆满，美妙的衣服覆盖周遍。菩萨的天冠和宝璎珞，都铺在那里的地上作为美好的庄严，栴檀摩尼普遍散布在其中，都舒展出离垢的美妙光明。宝华发出火焰，出现美妙的光明，光焰如同云彩照耀一切，散布这些美妙的华和各种宝物，普遍覆盖在地上作为庄严的装饰。密云兴起，布满十方世界，广大的光明没有穷尽，普遍到达十方一切国土，演说如来的甘露之法。在一切佛愿的摩尼宝之内，普遍显现无边广大的劫数，最殊胜的智慧者往昔所行的，在这个宝物中没有看不见的。那里的地所有的摩尼宝，一切佛刹都来进入，那些佛刹的每一个微尘，一切国土也进入其中。美妙的宝物庄严着华藏世界，菩萨游行遍布十方世界，演说大士的各种弘大誓愿，这是道场的自在之力。摩尼妙宝庄严的地，放出清净的光明，具备各种装饰，充满法界，等同于虚空，佛的力量自然地像这样显现。那些修行治理普贤愿，进入佛境界的大智慧之人，能知道在这个刹海中，像这样一切的各种神变。”*

尔时，普贤菩萨复告大众言：“诸佛子！此世界海大地中，有不可说佛刹微尘数香水海，一切妙宝庄严其底，妙香摩尼庄严其岸，毗卢遮那摩尼宝王以为其网；香水映彻，具众宝色，充满其中；种种宝华，旋布其上；栴檀细末，澄垽其下；演佛言音，放宝光明；无边菩萨，持种种盖，现神通力。一切世界所有庄严，悉于中现。十宝阶陛，行列分布；十宝栏楯，周匝围绕；四天下微尘数一切宝庄严芬陀利华，敷荣水中；不可说百千亿那由他数十宝尸罗幢，恒河沙数一切宝衣铃网幢，恒河沙数无边色相宝华楼阁，百千亿那由他数十宝莲华城，四天下微尘数众宝树林——宝焰摩尼以为其网，恒河沙数栴檀香，诸佛言音光焰摩尼，不可说百千亿那由他数众宝垣墙，悉共围绕，周遍严饰。”

*那时，普贤菩萨又告诉大众说：“各位佛子！在这个世界海的大地中，有不可说佛刹微尘那么多数的香水海，用一切美妙的宝物庄严它的底部，用美妙的香摩尼庄严它的岸边，用毗卢遮那摩尼宝王作为它的网；香水清澈透明，具备各种宝物的颜色，充满在其中；各种宝华，旋转地铺在上面；栴檀的细末，沉淀在下面；演说佛的言语音声，放出宝物光明；无边的菩萨，拿着各种伞盖，显现神通之力。一切世界所有的庄严，都在其中显现。十种宝阶，排列分布；十种宝栏杆，周遍围绕；四天下微尘那么多数的一切宝庄严的芬陀利华，在水中开放；不可说百千亿那由他那么多数的十宝尸罗幢，恒河沙数的一切宝衣铃网幢，恒河沙数的无边色相的宝华楼阁，百千亿那由他那么多数的十宝莲华城，四天下微尘那么多数的各种宝树林——用宝焰摩尼作为它的网，恒河沙数的栴檀香，诸佛言音光焰摩尼，不可说百千亿那由他那么多数的各种宝垣墙，都共同围绕，周遍庄严装饰。”*
