# 大方广佛华严经卷第十二
*大方广佛华严经卷第十二*

　　于阗国三藏实叉难陀奉　制译
　　*于阗国三藏法师实叉难陀奉皇帝命令翻译*

　　如来名号品第七
　　*如来名号品第七*

　　尔时，世尊在摩竭提国阿兰若法菩提场中，始成正觉，于普光明殿坐莲华藏师子之座，妙悟皆满，二行永绝；达无相法，住于佛住；得佛平等，到无障处；不可转法，所行无碍；立不思议，普见三世。与十佛刹微尘数诸菩萨俱，莫不皆是一生补处，悉从他方而共来集，普善观察诸众生界、法界、世界、涅槃界，诸业果报、心行次第、一切文义，世、出世间，有为、无为，过、现、未来。
　　*那时，世尊在摩竭提国的阿兰若法菩提场中，刚刚成就正觉，在普光明殿里坐在莲华藏师子之座上。他的奇妙觉悟都已圆满，两种执着永远断绝；通达了无相之法，安住于佛的境界；得到了佛的平等，到达了无障碍之处；证得了不可转动之法，所行无碍；建立了不可思议的功德，普遍见到过去、现在、未来三世。与十个佛刹微尘数那么多的菩萨们在一起，他们无一不是一生补处的菩萨，都从其他地方前来集会。他们都善于普遍观察各种众生界、法界、世界、涅槃界，各种业力果报、心念行为的次第、一切的文辞义理，世间、出世间，有为、无为，过去、现在、未来。*

　　时，诸菩萨作是思惟：“若世尊见愍我等，愿随所乐，开示佛刹、佛住、佛刹庄严、佛法性、佛刹清净、佛所说法、佛刹体性、佛威德、佛刹成就、佛大菩提。如十方一切世界诸佛世尊，为成就一切菩萨故，令如来种性不断故，救护一切众生故，令诸众生永离一切烦恼故，了知一切诸行故，演说一切诸法故，净除一切杂染故，永断一切疑网故，拔除一切希望故，灭坏一切爱着处故，说诸菩萨十住、十行、十回向、十藏、十地、十愿、十定、十通、十顶，及说如来地、如来境界、如来神力、如来所行、如来力、如来无畏、如来三昧、如来神通、如来自在、如来无碍、如来眼、如来耳、如来鼻、如来舌、如来身、如来意、如来辩才、如来智慧、如来最胜。愿佛世尊，亦为我说！”
　　*那时，各位菩萨这样思惟：“如果世尊怜悯我们，希望您随顺我们的心愿，开示佛刹、佛的安住、佛刹的庄严、佛的法性、佛刹的清净、佛所说的法、佛刹的体性、佛的威德、佛刹的成就、佛的大菩提。如同十方一切世界的诸佛世尊，为了成就一切菩萨，为了让如来的种性不断绝，为了救护一切众生，为了让众生永远脱离一切烦恼，为了了知一切的行为，为了演说一切的法，为了净化一切的杂染，为了永远断除一切的疑网，为了拔除一切的希望，为了灭坏一切爱执之处，而宣说诸位菩萨的十住、十行、十回向、十藏、十地、十愿、十定、十通、十顶，以及宣说如来的地、如来的境界、如来的神力、如来的所行、如来的力、如来的无畏、如来的三昧、如来的神通、如来的自在、如来的无碍、如来的眼、如来的耳、如来的鼻、如来的舌、如来的身、如来的意、如来的辩才、如来的智慧、如来的最胜。希望佛世尊，也为我们宣说！”*

　　尔时，世尊知诸菩萨心之所念，各随其类，为现神通。现神通已，东方过十佛刹微尘数世界，有世界，名：金色，佛号：不动智。彼世界中，有菩萨，名：文殊师利，与十佛刹微尘数诸菩萨俱，来诣佛所，到已作礼，即于东方化作莲华藏师子之座，结跏趺坐。南方过十佛刹微尘数世界，有世界，名：妙色，佛号：无碍智。彼有菩萨，名曰：觉首，与十佛刹微尘数诸菩萨俱，来诣佛所，到已作礼，即于南方化作莲华藏师子之座，结跏趺坐。西方过十佛刹微尘数世界，有世界，名：莲华色，佛号：灭暗智。彼有菩萨，名曰：财首，与十佛刹微尘数诸菩萨俱，来诣佛所，到已作礼，即于西方化作莲华藏师子之座，结跏趺坐。北方过十佛刹微尘数世界，有世界，名：薝卜华色，佛号：威仪智。彼有菩萨，名曰：宝首，与十佛刹微尘数诸菩萨俱，来诣佛所，到已作礼，即于北方化作莲华藏师子之座，结跏趺坐。东北方过十佛刹微尘数世界，有世界，名：优钵罗华色，佛号：明相智。彼有菩萨，名：功德首，与十佛刹微尘数诸菩萨俱，来诣佛所，到已作礼，即于东北方化作莲华藏师子之座，结跏趺坐。东南方过十佛刹微尘数世界，有世界，名：金色，佛号：究竟智。彼有菩萨，名：目首，与十佛刹微尘数诸菩萨俱，来诣佛所，到已作礼，即于东南方化作莲华藏师子之座，结跏趺坐。西南方过十佛刹微尘数世界，有世界，名：宝色，佛号：最胜智。彼有菩萨，名：精进首，与十佛刹微尘数诸菩萨俱，来诣佛所，到已作礼，即于西南方化作莲华藏师子之座，结跏趺坐。西北方过十佛刹微尘数世界，有世界，名：金刚色，佛号：自在智。彼有菩萨，名：法首，与十佛刹微尘数诸菩萨俱，来诣佛所，到已作礼，即于西北方化作莲华藏师子之座，结跏趺坐。下方过十佛刹微尘数世界，有世界，名：玻璃色，佛号：梵智。彼有菩萨，名：智首，与十佛刹微尘数诸菩萨俱，来诣佛所，到已作礼，即于下方化作莲华藏师子之座，结跏趺坐。上方过十佛刹微尘数世界，有世界，名：平等色，佛号：观察智。彼有菩萨，名：贤首，与十佛刹微尘数诸菩萨俱，来诣佛所，到已作礼，即于上方化作莲华藏师子之座，结跏趺坐。
　　*那时，世尊知道了菩萨们心中的念头，便随顺各类众生，为他们显现神通。显现神通之后，东方经过十个佛刹微尘数那么多的世界，有一个世界，名叫‘金色’，佛号是‘不动智’。那个世界中，有一位菩萨，名叫‘文殊师利’，与十个佛刹微尘数那么多的菩萨们一同，来到佛的所在地，到达后行礼，就在东方化现出莲华藏师子之座，结跏趺坐而坐。南方经过十个佛刹微尘数那么多的世界，有一个世界，名叫‘妙色’，佛号是‘无碍智’。那里有一位菩萨，名叫‘觉首’，与十个佛刹微尘数那么多的菩萨们一同，来到佛的所在地，到达后行礼，就在南方化现出莲华藏师子之座，结跏趺坐而坐。西方经过十个佛刹微尘数那么多的世界，有一个世界，名叫‘莲华色’，佛号是‘灭暗智’。那里有一位菩萨，名叫‘财首’，与十个佛刹微尘数那么多的菩萨们一同，来到佛的所在地，到达后行礼，就在西方化现出莲华藏师子之座，结跏趺坐而坐。北方经过十个佛刹微尘数那么多的世界，有一个世界，名叫‘薝卜华色’，佛号是‘威仪智’。那里有一位菩萨，名叫‘宝首’，与十个佛刹微尘数那么多的菩萨们一同，来到佛的所在地，到达后行礼，就在北方化现出莲华藏师子之座，结跏趺坐而坐。东北方经过十个佛刹微尘数那么多的世界，有一个世界，名叫‘优钵罗华色’，佛号是‘明相智’。那里有一位菩萨，名叫‘功德首’，与十个佛刹微尘数那么多的菩萨们一同，来到佛的所在地，到达后行礼，就在东北方化现出莲华藏师子之座，结跏趺坐而坐。东南方经过十个佛刹微尘数那么多的世界，有一个世界，名叫‘金色’，佛号是‘究竟智’。那里有一位菩萨，名叫‘目首’，与十个佛刹微尘数那么多的菩萨们一同，来到佛的所在地，到达后行礼，就在东南方化现出莲华藏师子之座，结跏趺坐而坐。西南方经过十个佛刹微尘数那么多的世界，有一个世界，名叫‘宝色’，佛号是‘最胜智’。那里有一位菩萨，名叫‘精进首’，与十个佛刹微尘数那么多的菩萨们一同，来到佛的所在地，到达后行礼，就在西南方化现出莲华藏师子之座，结跏趺坐而坐。西北方经过十个佛刹微尘数那么多的世界，有一个世界，名叫‘金刚色’，佛号是‘自在智’。那里有一位菩萨，名叫‘法首’，与十个佛刹微尘数那么多的菩萨们一同，来到佛的所在地，到达后行礼，就在西北方化现出莲华藏师子之座，结跏趺坐而坐。下方经过十个佛刹微尘数那么多的世界，有一个世界，名叫‘玻璃色’，佛号是‘梵智’。那里有一位菩萨，名叫‘智首’，与十个佛刹微尘数那么多的菩萨们一同，来到佛的所在地，到达后行礼，就在下方化现出莲华藏师子之座，结跏趺坐而坐。上方经过十个佛刹微尘数那么多的世界，有一个世界，名叫‘平等色’，佛号是‘观察智’。那里有一位菩萨，名叫‘贤首’，与十个佛刹微尘数那么多的菩萨们一同，来到佛的所在地，到达后行礼，就在上方化现出莲华藏师子之座，结跏趺坐而坐。*

　　尔时，文殊师利菩萨摩诃萨，承佛威力，普观一切菩萨众会而作是言：
　　*那时，文殊师利菩萨摩诃萨，承蒙佛的威力，普视一切菩萨众会而这样说道：*

　　“此诸菩萨甚为希有！诸佛子！佛国土不可思议，佛住、佛刹庄严、佛法性、佛刹清净、佛说法、佛出现、佛刹成就、佛阿耨多罗三藐三菩提皆不可思议。何以故？诸佛子！十方世界一切诸佛，知诸众生乐欲不同，随其所应，说法调伏，如是乃至等法界、虚空界。
　　*“这些菩萨真是非常稀有！各位佛子！佛的国土不可思议，佛的安住、佛刹的庄严、佛的法性、佛刹的清净、佛的说法、佛的出现、佛刹的成就、佛的无上正等正觉都不可思议。为什么呢？各位佛子！十方世界一切诸佛，知道众生的喜好和欲望不同，便随顺他们的情况，说法调伏他们，如此乃至遍及法界、虚空界。*

　　“诸佛子！如来于此娑婆世界诸四天下，种种身、种种名、种种色相、种种修短、种种寿量、种种处所、种种诸根、种种生处、种种语业、种种观察，令诸众生各别知见。
　　*“各位佛子！如来在这个娑婆世界的四个天下中，以各种不同的身体、各种不同的名称、各种不同的色相、各种不同的长短、各种不同的寿命、各种不同的处所、各种不同的根性、各种不同的出生地、各种不同的语言业、各种不同的观察，使得众生各自有不同的了知和见解。*

　　“诸佛子！如来于此四天下中，或名：一切义成，或名：圆满月，或名：师子吼，或名：释迦牟尼，或名：第七仙，或名：毗卢遮那，或名：瞿昙氏，或名：大沙门，或名：最胜，或名：导师……。如是等，其数十千，令诸众生各别知见。
　　*“各位佛子！如来在这四个天下中，有时名叫‘一切义成’，有时名叫‘圆满月’，有时名叫‘师子吼’，有时名叫‘释迦牟尼’，有时名叫‘第七仙’，有时名叫‘毗卢遮那’，有时名叫‘瞿昙氏’，有时名叫‘大沙门’，有时名叫‘最胜’，有时名叫‘导师’……。像这样有数十千种名称，使得众生各自有不同的了知和见解。*

　　“诸佛子！此四天下东，次有世界，名为：善护。如来于彼，或名：金刚，或名：自在，或名：有智慧，或名：难胜，或名：云王，或名：无诤，或名：能为主，或名：心欢喜，或名：无与等，或名：断言论……。如是等，其数十千，令诸众生各别知见。
　　*“各位佛子！这四个天下的东方，紧邻着一个世界，名叫‘善护’。如来在那里，有时名叫‘金刚’，有时名叫‘自在’，有时名叫‘有智慧’，有时名叫‘难胜’，有时名叫‘云王’，有时名叫‘无诤’，有时名叫‘能为主’，有时名叫‘心欢喜’，有时名叫‘无与等’，有时名叫‘断言论’……。像这样有数十千种名称，使得众生各自有不同的了知和见解。*

　　*“各位佛子！这四个天下的南方，紧邻着一个世界，名叫‘难忍’。如来在那里，有时名叫‘帝释’，有时名叫‘宝称’，有时名叫‘离垢’，有时名叫‘实语’，有时名叫‘能调伏’，有时名叫‘具足喜’，有时名叫‘大名称’，有时名叫‘能利益’，有时名叫‘无边’，有时名叫‘最胜’……。像这样有数十千种名称，使得众生各自有不同的了知和见解。*
