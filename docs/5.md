# 大方广佛华严经卷第五

*《大方广佛华严经》卷第五*

于阗国三藏实叉难陀奉 制译

*于阗国三藏法师实叉难陀奉皇帝命令翻译*

世主妙严品第一之五

*世主妙严品第一章之五*

复次，普贤菩萨摩诃萨，入不思议解脱门方便海，入如来功德海。所谓：有解脱门，名：严净一切佛国土调伏众生令究竟出离。有解脱门，名：普诣一切如来所修具足功德境界。有解脱门，名：安立一切菩萨地诸大愿海。有解脱门，名：普现法界微尘数无量身。有解脱门，名：演说遍一切国土不可思议数差别名。有解脱门，名：一切微尘中悉现无边诸菩萨神通境界。有解脱门，名：一念中现三世劫成坏事。有解脱门，名：示现一切菩萨诸根海各入自境界。有解脱门，名：能以神通力化现种种身遍无边法界。有解脱门，名：显示一切菩萨修行法次第门入一切智广大方便。

*其次，普贤菩萨摩诃萨，进入了不可思议的解脱法门的方便之海，进入了如来的功德之海。具体来说，有解脱门，名为：庄严清净一切佛国土，调伏众生，让他们究竟出离。有解脱门，名为：普遍前往一切如来之处，修行具足的功德境界。有解脱门，名为：安立一切菩萨地的各种大愿之海。有解脱门，名为：普遍显现法界微尘数量的无量身体。有解脱门，名为：演说遍及一切国土的不可思议数量的差别名号。有解脱门，名为：在一切微尘中都显现无边的诸菩萨神通境界。有解脱门，名为：在一念之中显现三世劫的成住坏空之事。有解脱门，名为：示现一切菩萨的各种根器之海，各自进入自己的境界。有解脱门，名为：能够用神通之力化现各种身体，遍及无边的法界。有解脱门，名为：显示一切菩萨修行之法的次第法门，进入一切智慧的广大方便。*

尔时，普贤菩萨摩诃萨，以自功德，复承如来威神之力，普观一切众会海已，即说颂言：

*那时，普贤菩萨摩诃萨，凭借自己的功德，又承受如来的威神之力，普遍观察一切众会之海后，就说偈颂道：*

“佛所庄严广大刹，　　等于一切微尘数，
　清净佛子悉满中，　　雨不思议最妙法。
　如于此会见佛坐，　　一切尘中悉如是，
　佛身无去亦无来，　　所有国土皆明现。
　显示菩萨所修行，　　无量趣地诸方便，
　及说难思真实理，　　令诸佛子入法界。
　出生化佛如尘数，　　普应群生心所欲，
　入深法界方便门，　　广大无边悉开演。
　如来名号等世间，　　十方国土悉充遍，
　一切方便无空过，　　调伏众生皆离垢。
　佛于一切微尘中，　　示现无边大神力，
　悉坐道场能演说，　　如佛往昔菩提行。
　三世所有广大劫，　　佛念念中皆示现，
　彼诸成坏一切事，　　不思议智无不了。
　佛子众会广无限，　　欲共测量诸佛地，
　诸佛法门无有边，　　能悉了知甚为难。
　佛如虚空无分别，　　等真法界无所依，
　化现周行靡不至，　　悉坐道场成正觉。
　佛以妙音广宣畅，　　一切诸地皆明了，
　普现一一众生前，　　尽与如来平等法。”

*“佛所庄严的广大刹土，等于一切微尘的数量，清净的佛弟子都充满其中，降下不可思议的最美妙的佛法。如同在这个法会中看见佛坐着，在一切微尘中也都是这样，佛身没有来也没有去，所有的国土都清晰地显现。显示菩萨所修行的，趣向无量地的各种方便法门，以及解说难以思议的真实道理，让各位佛弟子进入法界。产生化身佛如同微尘的数量，普遍感应众生的心愿，进入甚深法界的方便法门，广大无边都开示演说。如来的名号等同于世间，十方国土都充满了，一切方便法门都不会白费，调伏众生都让他们脱离尘垢。佛在一切微尘中，显现无边的大神通之力，都坐在道场中能够演说，如同佛往昔的菩提行。三世所有广大的劫数，佛在念念之中都显现出来，那些成住坏空的一切事情，不可思议的智慧没有不了知的。佛弟子的众会广大无限，想共同测量诸佛的境地，诸佛的法门没有边际，能够完全了知非常困难。佛如同虚空没有分别，等同于真实的法界而无所依靠，化现周遍流行，没有不到达的地方，都坐在道场中成就正觉。佛用美妙的声音广泛地宣说，一切各个地都清晰地明了，普遍显现在每一个众生面前，都给予他们与如来平等的法。”*

复次，净德妙光菩萨摩诃萨，得遍往十方菩萨众会庄严道场解脱门；普德最胜灯光照菩萨摩诃萨，得一念中现无尽成正觉门教化成熟不思议众生界解脱门；普光师子幢菩萨摩诃萨，得修习菩萨福德庄严出生一切佛国土解脱门；普宝焰妙光菩萨摩诃萨，得观察佛神通境界无迷惑解脱门；普音功德海幢菩萨摩诃萨，得于一众会道场中示现一切佛土庄严解脱门；普智光照如来境菩萨摩诃萨，得随逐如来观察甚深广大法界藏解脱门；普觉悦意声菩萨摩诃萨，得亲近承事一切诸佛供养藏解脱门；普清净无尽福威光菩萨摩诃萨，得出世一切神变广大加持解脱门；普宝髻华幢菩萨摩诃萨，得普入一切世间行出生菩萨无边行门解脱门；普相最胜光菩萨摩诃萨，得能于无相法界中出现一切诸佛境界解脱门。

*其次，净德妙光菩萨摩诃萨，得到了遍往十方菩萨众会庄严道场的解脱门；普德最胜灯光照菩萨摩诃萨，得到了一念之中显现无尽的成正觉之门，教化成熟不可思议的众生界的解脱门；普光师子幢菩萨摩诃萨，得到了修习菩萨福德庄严，产生一切佛国土的解脱门；普宝焰妙光菩萨摩诃萨，得到了观察佛的神通境界而没有迷惑的解脱门；普音功德海幢菩萨摩诃萨，得到在一个众会道场中示现一切佛土庄严的解脱门；普智光照如来境菩萨摩诃萨，得到了随逐如来观察甚深广大法界宝藏的解脱门；普觉悦意声菩萨摩诃萨，得到了亲近承事一切诸佛的供养宝藏的解脱门；普清净无尽福威光菩萨摩诃萨，得到了出世间一切神变广大加持的解脱门；普宝髻华幢菩萨摩诃萨，得到了普遍进入一切世间之行，产生菩萨无边行门的解脱门；普相最胜光菩萨摩诃萨，得到了能够在无相的法界中出现一切诸佛境界的解脱门。*

尔时，净德妙光菩萨摩诃萨，承佛威力，普观一切菩萨解脱门海已，即说颂言：

*那时，净德妙光菩萨摩诃萨，承受佛的威力，普遍观察一切菩萨的解脱门之海后，就说偈颂道：*

“十方所有诸国土，　　一刹那中悉严净，
　以妙音声转法轮，　　普遍世间无与等。
　如来境界无边际，　　一念法界悉充满，
　一一尘中建道场，　　悉证菩提起神变。
　世尊往昔修诸行，　　经于百千无量劫，
　一切佛刹皆庄严，　　出现无碍如虚空。
　佛神通力无限量，　　充满无边一切劫，
　假使经于无量劫，　　念念观察无疲厌。
　汝应观佛神通境，　　十方国土皆严净，
　一切于此悉现前，　　念念不同无量种。
　观佛百千无量劫，　　不得一毛之分限，
　如来无碍方便门，　　此光普照难思刹。
　如来往劫在世间，　　承事无边诸佛海，
　是故一切如川骛，　　咸来供养世所尊。
　如来出现遍十方，　　一一尘中无量土，
　其中境界皆无量，　　悉住无边无尽劫。
　佛于曩劫为众生，　　修习无边大悲海，
　随诸众生入生死，　　普化众会令清净。
　佛住真如法界藏，　　无相无形离诸垢，
　众生观见种种身，　　一切苦难皆消灭。”

*“十方世界所有的国土，在一刹那之间都庄严清净，用美妙的声音转动法轮，普遍于世间无人能比。如来的境界没有边际，一念之间法界都充满了，在每一个微尘中都建立道场，都证得菩提，兴起神通变化。世尊往昔修行各种行门，经历了百千无量的劫数，一切佛刹都庄严了，出现时无所障碍，如同虚空。佛的神通之力没有限量，充满无边的一切劫数，假使经历了无量的劫数，念念之间观察也不疲厌。你应该观察佛的神通境界，十方国土都庄严清净，一切都在这里显现出来，念念之间都不同，有无量的种类。观察佛百千无量的劫数，也得不到他一根毛发的分量，如来无碍的方便法门，这光明普遍照耀难以思议的刹土。如来往昔的劫数中在世间，承事了无边的诸佛之海，所以一切都像江河奔流一样，都来供养世间所尊敬的佛。如来出现遍及十方世界，在每一个微尘中都有无量的国土，其中的境界都无量，都安住在无边无尽的劫数中。佛在往昔的劫数中为了众生，修习了无边的大悲之海，随着众生进入生死轮回，普遍教化众会让他们清净。佛安住在真如法界的宝藏中，没有相状，没有形体，远离各种尘垢，众生观察到各种不同的身体，一切的苦难都消灭了。”*

复次，海月光大明菩萨摩诃萨，得出生菩萨诸地诸波罗蜜教化众生及严净一切佛国土方便解脱门；云音海光离垢藏菩萨摩诃萨，得念念中普入法界种种差别处解脱门；智生宝髻菩萨摩诃萨，得不可思议劫于一切众生前现清净大功德解脱门；功德自在王净光菩萨摩诃萨，得普见十方一切菩萨初诣道场时种种庄严解脱门；善勇猛莲华髻菩萨摩诃萨，得随诸众生根解海普为显示一切佛法解脱门；普智云日幢菩萨摩诃萨，得成就如来智永住无量劫解脱门；大精进金刚𪗇菩萨摩诃萨，得普入一切无边法印力解脱门；香焰光幢菩萨摩诃萨，得显示现在一切佛始修菩萨行乃至成就智慧聚解脱门；大明德深美音菩萨摩诃萨，得安住毗卢遮那一切大愿海解脱门；大福光智生菩萨摩诃萨，得显示如来遍法界甚深境界解脱门。

*其次，海月光大明菩萨摩诃萨，得到了产生菩萨的各个地、各种波罗蜜，教化众生以及庄严清净一切佛国土的方便解脱门；云音海光离垢藏菩萨摩诃萨，得到了在念念之中普遍进入法界各种不同之处的解脱门；智生宝髻菩萨摩诃萨，得到了在不可思议的劫数中，在一切众生面前显现清净大功德的解脱门；功德自在王净光菩萨摩诃萨，得到了普遍看见十方一切菩萨初到道场时的各种庄严的解脱门；善勇猛莲华髻菩萨摩诃萨，得到了随着众生根器和理解之海，普遍为他们显示一切佛法的解脱门；普智云日幢菩萨摩诃萨，得到了成就如来智慧，永远安住在无量劫中的解脱门；大精进金刚𪗇菩萨摩诃萨，得到了普遍进入一切无边法印之力的解脱门；香焰光幢菩萨摩诃萨，得到了显示现在一切佛从开始修行菩萨行，乃至成就智慧之聚的解脱门；大明德深美音菩萨摩诃萨，得到了安住在毗卢遮那佛一切大愿之海的解脱门；大福光智生菩萨摩诃萨，得到了显示如来遍布法界的甚深境界的解脱门。*

尔时，海月光大明菩萨摩诃萨，承佛威力，普观一切菩萨众庄严海已，即说颂言：

*那时，海月光大明菩萨摩诃萨，承受佛的威力，普遍观察一切菩萨众的庄严之海后，就说偈颂道：*

“诸波罗蜜及诸地，　　广大难思悉圆满，
　无量众生尽调伏，　　一切佛土皆严净。
　如佛教化众生界，　　十方国土皆充满，
　一念心中转法轮，　　普应群情无不遍。
　佛于无量广大劫，　　普现一切众生前，
　如其往昔广修治，　　示彼所行清净处。
　我睹十方无有余，　　亦见诸佛现神通，
　悉坐道场成正觉，　　众会闻法共围绕。
　广大光明佛法身，　　能以方便现世间，
　普随众生心所乐，　　悉称其根而雨法。
　真如平等无相身，　　离垢光明净法身，
　智慧寂静身无量，　　普应十方而演法。
　法王诸力皆清净，　　智慧如空无有边，
　悉为开示无遗隐，　　普使众生同悟入。
　如佛往昔所修治，　　乃至成于一切智，
　今放光明遍法界，　　于中显现悉明了。
　佛以本愿现神通，　　一切十方无不照，
　如佛往昔修治行，　　光明网中皆演说。
　十方境界无有尽，　　无等无边各差别，
　佛无碍力发大光，　　一切国土皆明显。”

*“各种波罗蜜和各个地，广大难以思议，都已圆满，无量的众生都已调伏，一切佛土都庄严清净。如同佛教化众生界，十方国土都充满了，在一念心中转动法轮，普遍感应众生的情感，没有不遍及的。佛在无量广大的劫数中，普遍显现在一切众生面前，如同他往昔广泛修行治理，向他们开示所行的清净之处。我看见十方世界没有余留，也看见诸佛显现神通，都坐在道场中成就正觉，众会听闻佛法，共同围绕。广大的光明佛法身，能够用方便法门显现在世间，普遍随着众生的心愿，都称合他们的根器而降下法雨。真如平等，无相之身，离垢光明，清净的法身，智慧寂静，身体无量，普遍感应十方而演说佛法。法王的各种力量都清净，智慧如同虚空没有边际，都为众生开示而没有遗漏和隐藏，普遍使众生都一同悟入。如同佛往昔所修行治理，乃至成就了一切智慧，现在放出光明遍及法界，在其中显现，都清晰地明了。佛用本来的誓愿显现神通，一切十方世界没有不照耀的，如同佛往昔修行治理的行为，在光明之网中都演说了。十方的境界没有穷尽，无与伦比，无边无际，各不相同，佛用无碍之力发出大光明，一切国土都清晰地显现。”*

　各起净愿修诸行，　　悉曾供养无量佛，
　能见如来真实体，　　及以一切诸神变。
　或有能见佛法身，　　无等无碍普周遍，
　所有无边诸法性，　　悉入其身无不尽。
　或有见佛妙色身，　　无边色相光炽然，
　随诸众生解不同，　　种种变现十方中。
　或见无碍智慧身，　　三世平等如虚空，
　普随众生心乐转，　　种种差别皆令见。
　或有能了佛音声，　　普遍十方诸国土，
　随诸众生所应解，　　为出言音无障碍。
　或见如来种种光，　　种种照耀遍世间，
　或有于佛光明中，　　复见诸佛现神通。
　或见佛海云光，　　从毛孔出色炽然，
　示现往昔修行道，　　令生深信入佛智。
　或见佛相福庄严，　　及见此福所从生，
　往昔修行诸度海，　　皆佛相中明了见。
　如来功德不可量，　　充满法界无边际，
　及以神通诸境界，　　以佛力故能宣说。”

*“无量的众生处在法会中，有各种的信解，心地清净，都能悟入如来的智慧，了达一切庄严的境界。各自生起清净的誓愿，修行各种行门，都曾供养过无量的佛，能看见如来真实的体性，以及一切的各种神通变化。或者有能看见佛的法身，无与伦比，无所障碍，普遍周遍，所有无边的各种法性，都进入他的身体而没有不穷尽的。或者有看见佛的美妙色身，无边的色相光明炽盛，随着众生的理解不同，在十方世界中各种变化显现。或者看见无碍的智慧之身，三世平等如同虚空，普遍随着众生的心意和喜好而转动，各种差别都让他们看见。或者有能了知佛的声音，普遍于十方各个国土，随着众生所应该理解的，为他们发出言语声音而没有障碍。或者看见如来各种的光明，各种照耀遍及世间，或者有在佛的光明中，又看见诸佛显现神通。或者看见佛的海云光，从毛孔中发出炽盛的色彩，示现往昔修行的道路，让他们生起深深的信心，进入佛的智慧。或者看见佛的相好被福德庄严，以及看见这个福德从何而生，往昔修行各种波罗蜜之海，都在佛的相好中清晰地看见。如来的功德不可限量，充满法界没有边际，以及神通的各种境界，凭借佛的力量所以能宣说。”*

尔时，华藏庄严世界海，以佛神力，其地一切六种、十八相震动。所谓：动、遍动、普遍动；起、遍起、普遍起；涌、遍涌、普遍涌；震、遍震、普遍震；吼、遍吼、普遍吼；击、遍击、普遍击。此诸世主，一一皆现不思议诸供养云，雨于如来道场众海。所谓：一切香华庄严云、一切摩尼妙饰云、一切宝焰华网云、无边种类摩尼宝圆光云、一切众色宝真珠藏云、一切宝栴檀香云、一切宝盖云、清净妙声摩尼王云、日光摩尼璎珞轮云、一切宝光明藏云、一切各别庄严具云……。如是等诸供养云，其数无量不可思议。此诸世主，一一皆现如是供养云，雨于如来道场众海，靡不周遍。如此世界中，一一世主，心生欢喜，如是供养；其华藏庄-严世界海中，一切世界所有世主，悉亦如是而为供养。其一切世界中，悉有如来坐于道场；一一世主，各各信解，各各所缘，各各三昧方便门，各各修习助道法，各各成就，各各欢喜，各各趣入，各各悟解诸法门，各各入如来神通境界，各各入如来力境界，各各入如来解脱门。如于此华藏世界海，十方尽法界、虚空界、一切世界海中，悉亦如是。

*那时，华藏庄严世界海，凭借佛的神力，那里的地都发生了六种、十八相的震动。它们是：动、遍动、普遍动；起、遍起、普遍起；涌、遍涌、普遍涌；震、遍震、普遍震；吼、遍吼、普遍吼；击、遍击、普遍击。这些世主，每一个都显现了不可思议的各种供养之云，降落在如来的道场众海之中。它们是：一切香华庄严云、一切摩尼妙饰云、一切宝焰华网云、无边种类摩尼宝圆光云、一切众色宝真珠藏云、一切宝栴檀香云、一切宝盖云、清净妙声摩尼王云、日光摩尼璎珞轮云、一切宝光明藏云、一切各别庄严具云……。像这样等等的供养云，它们的数量无量不可思议。这些世主，每一个都显现了像这样的供养云，降落在如来的道场众海之中，没有不周遍的。在这个世界中，每一个世主，心中生起欢喜，像这样地供养；在那个华藏庄严世界海中，一切世界所有的世主，也都像这样地进行供养。在一切世界中，都有如来坐在道场；每一个世主，各自有不同的信解，各自有不同的所缘，各自有不同的三昧方便之门，各自修习不同的助道之法，各自有不同的成就，各自有不同的欢喜，各自有不同的趣入，各自有不同的悟解之法门，各自进入如来的神通境界，各自进入如来的力之境界，各自进入如来的解脱之门。如同在这个华藏世界海，十方穷尽法界、虚空界、一切世界海中，也都像这样。*
