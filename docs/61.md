# 大方广佛华严经卷第六十一

*大方广佛华严经卷第六十一*

　　于阗国三藏实叉难陀奉　制译

*于阗国三藏法师实叉难陀奉诏翻译*

　　入法界品第三十九之二

*入法界品第三十九之二*

　　尔时，普贤菩萨摩诃萨普观一切菩萨众会，以等法界方便、等虚空界方便、等众生界方便，等三世、等一切劫、等一切众生业、等一切众生欲、等一切众生解、等一切众生根、等一切众生成熟时、等一切法光影方便，为诸菩萨，以十种法句开发、显示、照明、演说此师子频申三昧。何等为十？所谓：演说能示现等法界一切佛刹微尘中，诸佛出兴次第、诸刹成坏次第法句；演说能示现等虚空界一切佛刹中，尽未来劫赞叹如来功德音声法句；演说能示现等虚空界一切佛刹中，如来出世无量无边成正觉门法句；演说能示现等虚空界一切佛刹中，佛坐道场菩萨众会法句；演说于一切毛孔，念念出现等三世一切佛变化身充满法界法句；演说能令一身充满十方一切刹海，平等显现法句；演说能令一切诸境界中，普现三世诸佛神变法句；演说能令一切佛刹微尘中，普现三世一切佛刹微尘数佛种种神变经无量劫法句；演说能令一切毛孔出生三世一切诸佛大愿海音，尽未来劫开发化导一切菩萨法句；演说能令佛师子座量同法界，菩萨众会道场庄严等无差别，尽未来劫转于种种微妙法轮法句。佛子！此十为首，有不可说佛刹微尘数法句，皆是如来智慧境界。

*那时候，普贤菩萨摩诃萨普遍地观察一切菩萨的集会，用等同法界的方便、等同虚空界的方便、等同众生界的方便，等同三世、等同一切劫、等同一切众生业、等同一切众生欲、等同一切众生解、等同一切众生根、等同一切众生成熟之时、等同一切法光影的方便，为各位菩萨，用十种法句来开发、显示、照明、演说这个师子频申三昧。是哪十种呢？就是：演说能示现等同法界的一切佛刹微尘之中，诸佛出兴的次第、各种刹土生成和毁灭的次第的法句；演说能示现等同虚空界的一切佛刹之中，穷尽未来的劫数赞叹如来功德的声音的法句；演说能示现等同虚空界的一切佛刹之中，如来出世无量无边成就正等正觉之门的法句；演说能示现等同虚空界的一切佛刹之中，佛坐在道场，菩萨大众集会的法句；演说在一切毛孔之中，念念之间出现等同三世一切佛的变化之身，充满法界的法句；演说能让一个身体充满十方一切的刹海，平等地显现的法句；演说能让一切各种境界之中，普遍显现三世诸佛的神通变化的法句；演说能让一切佛刹的微尘之中，普遍显现三世一切佛刹微尘数那么多的佛的各种神通变化，经过无量的劫数的法句；演说能让一切毛孔产生三世一切诸佛宏大誓愿之海的声音，穷尽未来的劫数开发化导一切菩萨的法句；演说能让佛的师子之座的量等同法界，菩萨的集会、道场的庄严相等而没有差别，穷尽未来的劫数转动各种微妙法轮的法句。佛子啊！以这十种为首，有不可说的佛刹微尘数那么多的法句，都是如来的智慧境界。*

　　尔时，普贤菩萨欲重宣此义，承佛神力，观察如来，观察众会，观察诸佛难思境界，观察诸佛无边三昧，观察不可思议诸世界海，观察不可思议如幻法智，观察不可思议三世诸佛悉皆平等，观察一切无量无边诸言辞法，而说颂言：

*那时候，普贤菩萨想要重新宣说这个义理，承受着佛的神力，观察如来，观察大众，观察诸佛难以思议的境界，观察诸佛无边的三昧，观察不可思议的各个世界海，观察不可思议的如同幻术的法智，观察不可思议的三世诸佛都完全平等，观察一切无量无边的各种言辞之法，而说出偈颂道：*

“一一毛孔中，　　微尘数刹海，
　悉有如来坐，　　皆具菩萨众。
　一一毛孔中，　　无量诸刹海，
　佛处菩提座，　　如是遍法界。
　一一毛孔中，　　一切刹尘佛，
　菩萨众围绕，　　为说普贤行。
　佛坐一国土，　　充满十方界，
　无量菩萨云，　　咸来集其所。
　亿刹微尘数，　　菩萨功德海，
　俱从会中起，　　遍满十方界。
　悉住普贤行，　　皆游法界海，
　普现一切刹，　　等入诸佛会。
　安坐一切刹，　　听闻一切法；
　一一国土中，　　亿劫修诸行。
　菩萨所修行，　　普明法海行，
　入于大愿海，　　住佛境界地。
　了达普贤行，　　出生诸佛法，
　具佛功德海，　　广现神通事。
　身云等尘数，　　充遍一切刹，
　普雨甘露法，　　令众住佛道。”

*“每一个毛孔之中， 有微尘数那么多的刹海， 都有如来坐在其中， 都具足菩萨大众。 每一个毛孔之中， 有无量的各个刹海， 佛处在菩提座上， 像这样遍及法界。 每一个毛孔之中， 有一切刹土微尘数那么多的佛， 被菩萨大众围绕， 为他们解说普贤的行持。 佛坐在一个国土， 却能充满十方界， 无量的菩萨如同云彩， 都来集会到他那里。 亿万刹土微尘数那么多的， 菩萨的功德之海， 都从集会中生起， 遍满十方界。 都安住于普贤的行持， 都游行于法界之海， 普遍地显现在一切刹土， 平等地进入诸佛的集会。 安坐于一切刹土， 听闻一切的法； 在每一个国土之中， 经过亿劫修习各种行持。 菩萨所修行的， 是普明的法海之行， 进入于大的誓愿之海， 安住于佛的境界之地。 了达普贤的行持， 产生诸佛之法， 具足佛的功德之海， 广泛地显现神通之事。 身体之云等同微尘数， 充满遍及一切的刹土， 普遍地降下甘露之法， 让大众安住于佛道。”*

　　尔时，世尊欲令诸菩萨安住如来师子频申广大三昧故，从眉间白毫相放大光明，其光名：普照三世法界门，以不可说佛刹微尘数光明而为眷属，普照十方一切世界海诸佛国土。时，逝多林菩萨大众，悉见一切尽法界、虚空界一切佛刹一一微尘中，各有一切佛刹微尘数诸佛国土，种种名、种种色、种种清净、种种住处、种种形相。如是一切诸国土中，皆有大菩萨坐于道场师子座上成等正觉，菩萨大众前后围绕，诸世间主而为供养；或见于不可说佛刹量大众会中，出妙音声充满法界，转正法轮；或见在天宫殿、龙宫殿、夜叉宫殿，乾闼婆、阿修罗、迦楼罗、紧那罗、摩睺罗伽、人、非人等诸宫殿中，或在人间村邑聚落、王都大处，现种种姓、种种名、种种身、种种相、种种光明，住种种威仪，入种种三昧，现种种神变，或时自以种种言音，或令种种诸菩萨等在于种种大众会中种种言辞说种种法。

*那时候，世尊想要让各位菩萨安住于如来师子频申广大三昧的缘故，从眉间的白毫相放出大的光明，那个光名叫：普照三世法界门，以不可说的佛刹微尘数那么多的光明作为眷属，普遍照耀十方一切的世界海、各位的佛国土。这时候，逝多林的菩萨大众，都见到一切穷尽法界、虚空界的一切佛刹，在每一个微尘之中，各自都有一切佛刹微尘数那么多的诸佛国土，有各种各样的名字、各种各样的颜色、各种各样的清净、各种各样的住处、各种各样的形状。像这样的一切各个国土之中，都有大菩萨坐在道场的师子之座上成就正等正觉，有菩萨大众前后围绕，有各位世间主宰来进行供养；或者见到在不可说的佛刹量那么大的大众集会之中，发出美妙的声音充满法界，转动正法轮；或者见到在天的宫殿、龙的宫殿、夜叉的宫殿，乾闼婆、阿修罗、迦楼罗、紧那罗、摩睺罗伽、人、非人等的各种宫殿之中，或者在人间的村庄、城市、聚落、王都等大的处所，显现各种各样的姓氏、各种各样的名字、各种各样的身体、各种各样的相状、各种各样的光明，安住于各种各样的威仪，进入各种各样的三昧，显现各种各样的神通变化，有时候自己用各种各样的言语声音，或者让各种各样的各位菩萨等在各种各样的大众集会之中用各种各样的言辞解说各种各样的法。*

　　如此会中，菩萨大众见于如是诸佛如来甚深三昧大神通力；如是尽法界、虚空界，东、西、南、北、四维、上、下一切方海中，依于众生心想而住，始从前际至今现在，一切国土身、一切众生身、一切虚空道，其中一一毛端量处，一一各有微尘数刹种种业起次第而住，悉有道场菩萨众会，皆亦如是见佛神力，不坏三世，不坏世间，于一切众生心中现其影像，随一切众生心乐出妙言音，普入一切众会中，普现一切众生前，色相有别，智慧无异，随其所应开示佛法，教化调伏一切众生未曾休息。其有见此佛神力者，皆是毗卢遮那如来于往昔时善根摄受，或昔曾以四摄所摄，或是见闻忆念亲近之所成熟，或是往昔教其令发阿耨多罗三藐三菩提心，或是往昔于诸佛所同种善根，或是过去以一切智善巧方便教化成熟，是故皆得入于如来不可思议甚深三昧；尽法界、虚空界大神通力，或入法身，或入色身，或入往昔所成就行，或入圆满诸波罗蜜，或入庄严清净行轮，或入菩萨诸地，或入成正觉力，或入佛所住三昧无差别大神变，或入如来力、无畏智，或入佛无碍辩才海。

*在这个集会之中，菩萨大众见到像这样各位佛如来甚深的三昧和大的神通之力；像这样穷尽法界、虚空界，东、西、南、北、四维、上、下一切方向的海中，依靠众生的心想而安住，从过去的边际开始一直到现在，一切的国土之身、一切的众生之身、一切的虚空之道，其中每一个毛的顶端那么大的处所，每一个都各自有微尘数那么多的刹土，各种各样的业生起的次第而安住，都有道场和菩萨的集会，也都像这样见到佛的神力，不破坏三世，不破坏世间，在一切众生的心中显现它的影像，随顺一切众生的心之所乐而发出美妙的言语声音，普遍地进入一切的集会之中，普遍地显现在一切众生的面前，色相有差别，智慧却没有差异，随顺他们所应接受的来开示佛法，教化调伏一切众生不曾休息。那些见到这个佛神力的人，都是毗卢遮那如来在过去的时候被善根所摄受的，或者是过去曾经被四摄法所摄受的，或者是通过见、闻、忆念、亲近所成熟的，或者是过去被教导让他发起无上正等正觉之心的，或者是过去在各位佛的处所共同种植善根的，或者是过去被一切智慧的善巧方便所教化成熟的，所以都得以进入如来不可思议的甚深三昧；穷尽法界、虚空界的大神通之力，或者进入法身，或者进入色身，或者进入过去所成就的行持，或者进入圆满的各种波罗蜜，或者进入庄严清净的行轮，或者进入菩萨的各种地，或者进入成就正等正觉之力，或者进入佛所安住的三昧无差别的大神通变化，或者进入如来的力、无所畏惧的智慧，或者进入佛无碍的辩才之海。*

*那时候，尊者舍利弗承受着佛的神力，见到文殊师利菩萨，与各位菩萨大众庄严地，走出逝多林，前往南方，在人间游行；他这样想：“我现在应当与文殊师利一同前往南方。”这时候，尊者舍利弗与六千比丘，前后围绕着，从自己的住处出来，来到佛的处所，顶礼佛足，详细地禀告世尊；世尊听许之后，向右围绕三圈，辞别退下而去，前往文殊师利的处所。这六千比丘是舍利弗自己所共同居住的，出家没有多久，就是：海觉比丘、善生比丘、福光比丘、大童子比丘、电生比丘、净行比丘、天德比丘、君慧比丘、梵胜比丘、寂慧比丘……。像这样等等，他们的数目有六千，都曾经供养过无量的诸佛，深深地种植了善根，理解力广大，信心的眼睛明了透彻，他们的心宽广博大，能观察佛的境界，了达法的本性，饶益众生，常常乐于勤奋追求诸佛的功德，都是被文殊师利说法教化所成就的。*

　　尔时，尊者舍利弗在行道中观诸比丘，告海觉言：“海觉！汝可观察文殊师利菩萨清净之身相好庄严，一切天人莫能思议。汝可观察文殊师利圆光映彻，令无量众生发欢喜心。汝可观察文殊师利光网庄严，除灭众生无量苦恼。汝可观察文殊师利众会具足，皆是菩萨往昔善根之所摄受。汝可观察文殊师利所行之路，左右八步，平坦庄严。汝可观察文殊师利所住之处，周回十方常有道场随逐而转。汝可观察文殊师利所行之路，具足无量福德庄严，左右两边有大伏藏，种种珍宝自然而出。汝可观察文殊师利曾供养佛，善根所流，一切树间出庄严藏。汝可观察文殊师利，诸世间主雨供具云，顶礼恭敬以为供养。汝可观察文殊师利，十方一切诸佛如来将说法时，悉放眉间白毫相光来照其身，从顶上入。”

*那时候，尊者舍利弗在行走的道路中观察各位比丘，告诉海觉说：“海觉！你可以观察文殊师利菩萨清净的身体、相好庄严，一切的天人不能思议。你可以观察文殊师利圆光映彻，让无量的众生发起欢喜心。你可以观察文殊师利光网的庄严，除灭众生无量的苦恼。你可以观察文殊师利集会的具足，都是被菩萨过去善根所摄受的。你可以观察文殊师利所行的道路，左右八步，平坦庄严。你可以观察文殊师利所住的处所，周围十方常常有道场随从而转。你可以观察文殊师利所行的道路，具足无量的福德庄严，左右两边有大的地下宝藏，各种各样的珍宝自然地生出。你可以观察文殊师利曾经供养佛，由善根所流出，一切的树之间生出庄严的宝藏。你可以观察文殊师利，各位世间主宰降下供养之具的云，顶礼恭敬来作为供养。你可以观察文殊师利，十方一切的诸佛如来将要说法的时候，都放出眉间的白毫相光来照耀他的身体，从头顶上进入。”*

　　尔时，尊者舍利弗为诸比丘称扬赞叹、开示演说文殊师利童子有如是等无量功德具足庄严。彼诸比丘闻是说已，心意清净，信解坚固，喜不自持，举身踊跃，形体柔软，诸根悦豫，忧苦悉除，垢障咸尽，常见诸佛，深求正法，具菩萨根，得菩萨力，大悲大愿皆自出生，入于诸度甚深境界，十方佛海常现在前，于一切智深生信乐；即白尊者舍利弗言：“唯愿大师将引我等，往诣于彼胜人之所。”时，舍利弗即与俱行，至其所已，白言：“仁者！此诸比丘，愿得奉觐。”

*那时候，尊者舍利弗为各位比丘称扬赞叹、开示演说文殊师利童子有像这样等等无量的功德具足庄严。那些比丘听了这个解说之后，心意清净，信解坚固，欢喜得不能自己，全身踊跃，形体柔软，各种根都喜悦豫悦，忧愁和痛苦都全部除掉，污垢和障碍都全部穷尽，常常见到诸佛，深深地追求正法，具足菩萨的根性，得到菩萨的力量，大的悲心和大的誓愿都自己产生，进入于各种度（波罗蜜）的甚深境界，十方的佛海常常显现在面前，对于一切智慧深深地生起信受和喜乐；随即对尊者舍利弗说：“只愿大师带领我们，前往那位殊胜之人的处所。”这时候，舍利弗就与他们一同前往，到了那里之后，禀告说：“仁者！这些比丘，愿意得以奉见。”*

　　尔时，文殊师利童子，无量自在菩萨围绕并其大众，如象王回观诸比丘。时，诸比丘顶礼其足，合掌恭敬，作如是言：“我今奉见，恭敬礼拜，及余所有一切善根。唯愿仁者文殊师利、和尚舍利弗、世尊释迦牟尼，皆悉证知！如仁所有如是色身、如是音声、如是相好、如是自在，愿我一切悉当具得。”

*那时候，文殊师利童子，被无量自在的菩萨围绕着以及他的大众，如同象王回头观察各位比丘。这时候，各位比丘顶礼他的脚，合掌恭敬，这样说：“我们现在奉见您，恭敬地礼拜，以及其余所有的一切善根。只愿仁者文殊师利、和尚舍利弗、世尊释迦牟尼，都全部证知！如同仁者所拥有的这样的色身、这样的声音、这样的相好、这样的自在，愿我们一切都将要具足得到。”*

　　尔时，文殊师利菩萨告诸比丘言：

*那时候，文殊师利菩萨告诉各位比丘说：*

　　“比丘！若善男子、善女人，成就十种趣大乘法，则能速入如来之地，况菩萨地！何者为十？所谓：积集一切善根，心无疲厌。见一切佛承事供养，心无疲厌。求一切佛法，心无疲厌。行一切波罗蜜，心无疲厌。成就一切菩萨三昧，心无疲厌。次第入一切三世，心无疲厌。普严净十方佛刹，心无疲厌。教化调伏一切众生，心无疲厌。于一切刹一切劫中成就菩萨行，心无疲厌。为成熟一众生故，修行一切佛刹微尘数波罗蜜，成就如来十力；如是次第，为成熟一切众生界，成就如来一切力，心无疲厌。

*“比丘！如果善男子、善女人，成就了十种趣向大乘的法，就能迅速地进入如来的境地，何况是菩萨的境地呢！是哪十种呢？就是：积集一切的善根，内心没有疲倦和厌烦。见到一切的佛都承事供养，内心没有疲倦和厌烦。追求一切的佛法，内心没有疲倦和厌烦。修行一切的波罗蜜，内心没有疲倦和厌烦。成就一切菩萨的三昧，内心没有疲倦和厌烦。次第地进入一切的三世，内心没有疲倦和厌烦。普遍地庄严清净十方的佛刹，内心没有疲倦和厌烦。教化调伏一切的众生，内心没有疲倦和厌烦。在一切的刹土、一切的劫数之中成就菩萨的行持，内心没有疲倦和厌烦。为了成熟一个众生的缘故，修行一切佛刹微尘数那么多的波罗蜜，成就如来的十力；像这样次第地，为了成熟一切的众生界，成就如来的一切力量，内心没有疲倦和厌烦。*

　　“比丘！若善男子、善女人，成就深信，发此十种无疲厌心，则能长养一切善根，舍离一切诸生死趣，超过一切世间种姓，不堕声闻、辟支佛地，生一切如来家，具一切菩萨愿，学习一切如来功德，修行一切菩萨诸行，得如来力，摧伏众魔及诸外道，亦能除灭一切烦恼，入菩萨地，近如来地。”

*“比丘！如果善男子、善女人，成就了深切的信心，发起这十种没有疲倦厌烦的心，就能长养一切的善根，舍离一切各种生死的趣向，超过一切世间的种姓，不堕入声闻、辟支佛的境地，出生在一切如来的家中，具足一切菩萨的誓愿，学习一切如来的功德，修行一切菩萨的各种行持，得到如来的力量，摧伏众魔以及各位外道，也能除灭一切的烦恼，进入菩萨的境地，接近如来的境地。”*

　　时，诸比丘闻此法已，则得三昧，名：无碍眼见一切佛境界。得此三昧故，悉见十方无量无边一切世界诸佛如来，及其所有道场众会；亦悉见彼十方世界一切诸趣所有众生；亦悉见彼一切世界种种差别；亦悉见彼一切世界所有微尘；亦悉见彼诸世界中，一切众生所住宫殿，以种种宝而为庄严；及亦闻彼诸佛如来种种言音演说诸法文辞训释，悉皆解了；亦能观察彼世界中一切众生诸根心欲；亦能忆念彼世界中一切众生前后十生；亦能忆念彼世界中过去、未来各十劫事；亦能忆念彼诸如来十本生事、十成正觉、十转法轮、十种神通、十种说法、十种教诫、十种辩才；又即成就十千菩提心、十千三昧、十千波罗蜜，悉皆清净；得大智慧圆满光明，得菩萨十神通，柔软微妙，住菩提心，坚固不动。

*那时候，各位比丘听了这个法之后，就得到了名为“无碍眼见一切佛境界”的三昧。因为得到这个三昧，所以都见到十方无量无边一切世界的诸佛如来，以及他们所有的道场和集会；也全部见到那个十方世界一切各种趣中所有的众生；也全部见到那个一切世界各种各样的差别；也全部见到那个一切世界所有的微尘；也全部见到那些世界之中，一切众生所住的宫殿，用各种各样的宝物来作为庄严；并且也听闻那些诸佛如来用各种各样的言语声音演说各种法的文辞和解释，都全部理解了；也能观察那个世界中一切众生的各种根性和心中所想的欲望；也能回忆起那个世界中一切众生前后十次的出生；也能回忆起那个世界中过去、未来各十个劫数的事情；也能回忆起那些各位如来的十件本生故事、十次成就正等正觉、十次转动法轮、十种神通、十种说法、十种教诫、十种辩才；又随即成就了一万种菩提心、一万种三昧、一万种波罗蜜，都全部清净；得到大智慧的圆满光明，得到菩萨的十种神通，柔软微妙，安住于菩提心，坚固不动。*

　　尔时，文殊师利菩萨劝诸比丘住普贤行；住普贤行已，入大愿海；入大愿海已，成就大愿海。以成就大愿海故，心清净；心清净故，身清净；身清净故，身轻利；身清净、轻利故，得大神通无有退转；得此神通故，不离文殊师利足下，普于十方一切佛所悉现其身，具足成就一切佛法。

*那时候，文殊师利菩萨劝各位比丘安住于普贤的行持；安住于普贤的行持之后，进入大的誓愿之海；进入大的誓愿之海之后，成就了大的誓愿之海。因为成就了大的誓愿之海，所以心清净；因为心清净，所以身清净；因为身清净，所以身体轻快便利；因为身清净、轻快便利，所以得到大的神通而没有退转；因为得到这个神通，所以不离开文殊师利的脚下，却能普遍地在十方一切佛的处所都显现他的身体，具足地成就一切的佛法。*
