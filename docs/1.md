# 大方广佛华严经卷第一

*《大方广佛华严经》卷第一*

于阗国三藏实叉难陀奉 制译

*于阗国三藏法师实叉难陀奉皇帝命令翻译*

世主妙严品第一之一

*世主妙严品第一章之一*

如是我闻：

*我是这样听说的：*

一时，佛在摩竭提国阿兰若法菩提场中，始成正觉。其地坚固，金刚所成；上妙宝轮，及众宝华、清净摩尼，以为严饰；诸色相海，无边显现；摩尼为幢，常放光明，恒出妙音，众宝罗网，妙香华缨，周匝垂布；摩尼宝王，变现自在，雨无尽宝及众妙华分散于地；宝树行列，枝叶光茂。佛神力故，令此道场一切庄严于中影现。其菩di树高显殊特：金刚为身，琉璃为干；众杂妙宝以为枝条；宝叶扶疏，垂荫如云；宝华杂色，分枝布影，复以摩尼而为其果，含辉发焰，与华间列。其树周围咸放光明，于光明中雨摩尼宝，摩尼宝内，有诸菩萨，其众如云，俱时出现。又以如来威神力故，其菩提树恒出妙音，说种种法，无有尽极。如来所处宫殿楼阁，广博严丽充遍十方，众色摩尼之所集成，种种宝华以为庄校；诸庄严具流光如云，从宫殿间萃影成幢。无边菩萨道场众会咸集其所，以能出现诸佛光明不思议音。摩尼宝王而为其网，如来自在神通之力所有境界皆从中出；一切众生居处屋宅，皆于此中现其影像。又以诸佛神力所加，一念之间，悉包法界。其师子座，高广妙好：摩尼为台，莲华为网，清净妙宝以为其轮，众色杂华而作璎珞。堂榭、楼阁、阶砌、户牖，凡诸物像，备体庄严；宝树枝果，周回间列。摩尼光云，互相照耀；十方诸佛，化现珠玉；一切菩萨髻中妙宝，悉放光明而来莹烛。复以诸佛威神所持，演说如来广大境界，妙音遐畅，无处不及。

*那时候，佛祖在摩竭提国的阿兰若法菩提道场中，刚刚证得无上正等正觉。那地方十分坚固，是由金刚构成的；有上等的妙宝轮，以及各种宝华、清净的摩尼宝珠，作为庄严的装饰；各种色相如海，无边无际地显现出来；摩尼宝珠作为旗幢，常常放出光明，不断发出美妙的声音；各种宝物结成的罗网，美妙的香华缨络， चारों周围都垂挂着；摩尼宝王，能够随意变现，降下无尽的宝物和各种美妙的华，散布在地上；宝树排列成行，枝叶光鲜茂盛。因为佛的威神之力，使这个道场中所有的庄严景象都在其中映现出来。那棵菩提树高大而又与众不同：金刚作为树身，琉璃作为树干；各种奇妙的宝物构成枝条；宝叶扶疏，垂下的树荫像云一样；宝华色彩斑斓，分枝布影；又用摩尼宝珠作为果实，含着光辉，发出火焰，与华交错排列。树的周围都放出光明，光明中降下摩尼宝珠，摩尼宝珠里面，有许多菩萨，他们的数量像云一样多，同时出现。又因为如来的威神之力，那棵菩提树常常发出美妙的声音，说各种各样的佛法，没有穷尽。如来所居住的宫殿楼阁，宽广庄严华丽，遍布十方世界，是由各种颜色的摩尼宝珠构成的，各种宝华作为庄严的点缀；各种庄严的器具流光如云，从宫殿之间汇聚的影子形成旗幢。无边的菩萨和道场的众会都聚集在那里，因为能够出现诸佛的光明和不可思议的声音。摩尼宝王作为它的罗网，如来运用自在神通之力所显现的一切境界都从中出来；一切众生居住的房屋，都在这里显现出它们的影像。又因为诸佛神力的加持，在一念之间，就完全包容了整个法界。那个师子座，高大、宽广、奇妙、美好：摩尼宝珠作为台，莲华作为网，清净的妙宝作为它的轮，各种颜色的杂华作成璎珞。堂、榭、楼阁、台阶、门窗，所有这些物像，都具备庄严的形体；宝树的枝果，周回交错排列。摩尼宝珠的光云，互相照耀；十方诸佛，化现出珠玉；一切菩萨发髻中的妙宝，都放出光明来照耀。又因为诸佛威神的加持，演说如来广大的境界，美妙的声音传得很远，没有地方听不到。*

尔时，世尊处于此座，于一切法成最正觉，智入三世悉皆平等，其身充满一切世间，其音普顺十方国土。譬如虚空具含众像，于诸境界无所分别；又如虚空普遍一切，于诸国土平等随入。身恒遍坐一切道场，菩萨众中威光赫奕，如日轮出，照明世界。三世所行，众福大海，悉已清净，而恒示生诸佛国土。无边色相，圆满光明，遍周法界，等无差别；演一切法，如布大云。一一毛端，悉能容受一切世界而无障碍，各现无量神通之力，教化调伏一切众生；身遍十方而无来往，智入诸相，了法空寂。三世诸佛所有神变，于光明中靡不咸睹；一切佛土不思议劫所有庄严，悉令显现。

*那时，世尊坐在这个座位上，对一切法都证得了最正确的觉悟，智慧进入过去、现在、未来三世，完全平等，他的身体充满了一切世间，他的声音普遍顺应十方国土。譬如虚空包含万象，对于各种境界没有分别；又如虚空遍及一切，对于各个国土平等地随入。身体永远遍坐一切道场，在菩萨大众中威光显赫，像太阳出来，照亮世界。过去、现在、未来三世所行的，众多的福德像大海一样，都已清净，而又常常示现在诸佛的国土中出生。无边的色相，圆满的光明，遍布整个法界，平等没有差别；演说一切法，像布施大片的云彩。每一个毛孔的顶端，都能容纳一切世界而没有障碍，各自显现无量的神通之力，教化调伏一切众生；身体遍布十方而没有来往，智慧进入各种相，了悟了法的空寂。过去、现在、未来三世诸佛所有的神通变化，在光明中没有不全看见的；一切佛土不可思议的劫数中所有的庄严，都让它们显现出来。*

有十佛世界微尘数菩萨摩诃萨所共围绕，其名曰：普贤菩萨摩诃萨、普德最胜灯光照菩萨摩诃萨、普光师子幢菩萨摩诃萨、普宝焰妙光菩萨摩诃萨、普音功德海幢菩萨摩诃萨、普智光照如来境菩萨摩诃萨、普宝髻华幢菩萨摩诃萨、普觉悦意声菩萨摩诃萨、普清净无尽福光菩萨摩诃萨、普光明相菩萨摩诃萨、海月光大明菩萨摩诃萨、云音海光无垢藏菩萨摩诃萨、功德宝髻智生菩萨摩诃萨、功德自在王大光菩萨摩诃萨、善勇猛莲华髻菩萨摩诃萨、普智云日幢菩萨摩诃萨、大精进金刚𪗇菩萨摩诃萨、香焰光幢菩萨摩诃萨、大明德深美音菩萨摩诃萨、大福光智生菩萨摩诃萨……。如是等而为上首，有十佛世界微尘数。此诸菩萨，往昔皆与毗卢遮那如来共集善根，修菩萨行；皆从如来善根海生，诸波罗蜜悉已圆满；慧眼明彻，等观三世；于诸三昧，具足清净；辩才如海，广大无尽；具佛功德，尊严可敬；知众生根，如应化伏；入法界藏，智无差别；证佛解脱，甚深广大；能随方便，入于一地，而以一切愿海所持，恒与智俱尽未来际；了达诸佛希有广大秘密之境，善知一切佛平等法，已践如来普光明地，入于无量三昧海门；于一切处，皆随现身；世法所行，悉同其事；总持广大，集众法海；辩才善巧，转不退轮；一切如来功德大海，咸入其身；一切诸佛所在国土，皆随愿往；已曾供养一切诸佛，无边际劫，欢喜无倦；一切如来得菩提处，常在其中，亲近不舍；恒以所得普贤愿海，令一切众生智身具足。成就如是无量功德。

*有十个佛世界微尘那么多的菩萨摩诃萨共同围绕着，他们的名字是：普贤菩萨摩诃萨、普德最胜灯光照菩萨摩诃萨、普光师子幢菩萨摩诃萨、普宝焰妙光菩萨摩诃萨、普音功德海幢菩萨摩诃萨、普智光照如来境菩萨摩诃萨、普宝髻华幢菩萨摩诃萨、普觉悦意声菩萨摩诃萨、普清净无尽福光菩萨摩诃萨、普光明相菩萨摩诃萨、海月光大明菩萨摩诃萨、云音海光无垢藏菩萨摩诃萨、功德宝髻智生菩萨摩诃萨、功德自在王大光菩萨摩诃萨、善勇猛莲华髻菩萨摩诃萨、普智云日幢菩萨摩诃萨、大精进金刚𪗇菩萨摩诃萨、香焰光幢菩萨摩诃萨、大明德深美音菩萨摩诃萨、大福光智生菩萨摩诃萨……。像这样等等作为上首，有十个佛世界微尘那么多的菩萨。这些菩萨，在往昔都和毗卢遮那如来一起积集善根，修行菩萨道；都从如来的善根海中出生，各种波罗蜜都已经圆满；慧眼明澈，平等地观察过去、现在、未来三世；对于各种三昧，都完全清净；辩才像大海一样，广大无尽；具备佛的功德，尊严可敬；知道众生的根性，根据情况来教化他们；进入法界藏，智慧没有差别；证得佛的解脱，非常深广；能够随机应变，进入某一个境界，而又被一切愿海所加持，永远和智慧一起直到未来的尽头；了达诸佛稀有广大的秘密境界，善于知道一切佛的平等法，已经踏上如来普光明地，进入无量的三昧海门；在一切地方，都随时显现身体；世间法的修行，都和他们一样；总持广大，汇集众多的法海；辩才善巧，转动不退转的法轮；一切如来的功德大海，都进入他们的身体；一切诸佛所在的国土，都随着愿望前往；曾经供养过一切诸佛，经历了无边无际的劫数，欢喜而不疲倦；一切如来证得菩提的地方，常常在其中，亲近而不舍弃；永远用所得到的普贤愿海，让一切众生的智慧身体都具足。成就了这样无量的功德。*

复有佛世界微尘数执金刚神，所谓：妙色那罗延执金刚神、日轮速疾幢执金刚神、须弥华光执金刚神、清净云音执金刚神、诸根美妙执金刚神、可爱乐光明执金刚神、大树雷音执金刚神、师子王光明执金刚神、密焰胜目执金刚神、莲华光摩尼髻执金刚神……。如是等而为上首，有佛世界微尘数，皆于往昔无量劫中恒发大愿，愿常亲近供养诸佛；随愿所行，已得圆满，到于彼岸；积集无边清净福业，于诸三昧所行之境悉已明达；获神通力，随如来住，入不思议解脱境界；处于众会，威光特达，随诸众生所应现身而示调伏；一切诸佛化形所在，皆随化往；一切如来所住之处，常勤守护。
